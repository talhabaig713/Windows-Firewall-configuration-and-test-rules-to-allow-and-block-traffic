# Windows-Firewall-configuration-and-test-rules-to-allow-and-block-traffic
Configure and tset basic firewall rules to allow or block traffic:
Configuring and testing the Windows Firewall involves managing rules for incoming and outgoing network traffic, as well as verifying that the firewall is functioning as intended. The tool used for advanced settings is called Windows Defender Firewall with Advanced Security. 
# Configure Windows Firewall with Advanced Security
Access the firewall settings
You can access the advanced settings through a few methods:

    Via Search: In the Windows search bar, type wf.msc and press Enter.
    Via Control Panel: Go to Control Panel > System and Security > Windows Defender Firewall, and then click Advanced settings.
